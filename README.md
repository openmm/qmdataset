# Small-molecule/Protein Interaction Chemical Energies (SPICE)-dataset
A collection of QM data for training potential functions

- QC specification: 
    - spec_name: `spice_default`
    - method: wB97M-D3BJ
    - basis: def2-TZVPPD
    - program: psi4
  
 - SCF properties:
    - dipole
    - quadrupole
    - wiberg_lowdin_indices
    - mayer_indices
    - mbis_charges
